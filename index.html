Омлет с помидорами

Ингредиенты:

  • Яйца - 3 шт.
  • Помидоры - 1 шт.
  • Соль, перец - по вкусу
  • Масло для жарки

Инструкции:

  1. Разбейте яйца в миску, добавьте соль и перец, взбейте вилкой.
  2. Нарежьте помидоры кубиками.
  3. Разогрейте сковороду с маслом на среднем огне.
  4. Вылейте в сковороду взбитые яйца, добавьте помидоры.
  5. Жарьте омлет до готовности с обеих сторон.
  6. Подавайте горячим.

Гранола с йогуртом и фруктами

Ингредиенты:

  • Гранола - 1 стакан
  • Натуральный йогурт - 1 стакан
  • Фрукты (ягоды, банан, яблоко) - по вкусу

Инструкции:

  1. Нарежьте фрукты.
  2. В большой чашке или стакане положите слой гранолы.
  3. Добавьте слой нарезанных фруктов.
  4. Залейте йогуртом.
  5. Повторите слои, заканчивая йогуртом сверху.
  6. Подавайте сразу.

Тосты с авокадо и яйцом

Ингредиенты:

  • Хлеб для тостов - 2 ломтика
  • Авокадо - 1 шт.
  • Яйца - 2 шт.
  • Соль, перец - по вкусу
  • Масло для жарки

Инструкции:

  1. Разрежьте авокадо пополам, удалите косточку и выложите мякоть ложкой в миску. Растолките вилкой.
  2. Разогрейте сковороду с маслом на среднем огне.
  3. Обжарьте тосты до золотистой корки.
  4. Жарьте яйца на сковороде до желаемой степени прожарки.
  5. Намажьте пюре из авокадо на тосты, выложите сверху яйца.
  6. Посолите и поперчите по вкусу.
  7. Подавайте горячими.
